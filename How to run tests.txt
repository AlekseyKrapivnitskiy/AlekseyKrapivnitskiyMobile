To run native test:
    1.Open http://mobilecloud.epam.com/.
    2.Open mobile phone with idiu from native.properties.
    3.Install ContactManager.apk.
    4.Click on nativetest.xml file by right mouse button.
    5.Click on: Run '.../src/test/resources/nativetest.xml'.

To run web test:
    1.Click on webtest.xml file by right mouse button.
    2.Click on: Run '.../src/test/resources/webtest.xml'.
